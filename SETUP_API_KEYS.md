## â“ Troubleshooting

### "No API key configured"
- Make sure you added at least one API key to `backend/.env`
- Check for typos in the key
- Ensure no extra spaces around the `=` sign

### "Invalid API key"
- Verify the key is correct in your provider's console
- Make sure the key hasn't been deleted or expired
- Try generating a new key

### "Application won't start"
- Check that `backend/.env` file exists
- Verify at least one API key is set
- Check Docker logs: `docker-compose logs backend`
